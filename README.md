# Pathing Map Align Plugin - Blish HUD Module
Align the Pathing with map manually.  
Say goodbye to incomprehensible DPI calculation.

Must be enabled after the Pathing module is enabled.
## Features
* Align trails and markers
* Adjust mini map size

## Usage
1. Open the module setting. 
2. Adjust various parameters.

## Download
From [Releases](https://github.com/cy-sp-howard/PathingMapAlignPlugin/releases) page.
## Install Guide
1. Right-click Blish-HUD icon from system tray where bottom right.
2. Left-click "Open settings folder".
3. Paste [PathingMapAlignPlugin.bhm](https://github.com/cy-sp-howard/PathingMapAlignPlugin/releases) file into "modules" folder.
4. Restart Blish-HUD.
5. Click the Blish-HUD icon in the game then Left-click the `PathingMapAlignPlugin` tab under `Manage Modules` section.
6. Click the `Enable` button for enable `PathingMapAlignPlugin` module.

More detail from Blish-HUD Offical [Tutorial](https://blishhud.com/docs/user/guides/installing-modules/#manually-installing-modules).
